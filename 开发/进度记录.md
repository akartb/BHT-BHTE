**⚠️ 编译环境限制说明 (2026-03-27)：**

由于沙盒环境权限限制，无法访问 Visual Studio 编译器 (cl.exe)，需要在本地 Windows 机器上使用以下步骤编译：

```batch
# 1. 打开 "x64 Native Tools Command Prompt for VS 2022"
#    或运行: "%VSINSTALLDIR%\VC\Auxiliary\Build\vcvarsall.bat" x64

# 2. 配置与编译
cd /d D:\work0\BHT\BHTWallet
if exist build rmdir /s /q build
mkdir build
cd build
cmake .. -G "NMake Makefiles" -DCMAKE_BUILD_TYPE=Release -DBUILD_GUI=OFF -DBUILD_CLI=ON
nmake

# 3. 测试 CLI
bin\bht-wallet-cli.exe version
bin\bht-wallet-cli.exe help
```

或使用 Visual Studio generator：
```batch
cmake .. -G "Visual Studio 17 2022" -A x64 -DCMAKE_BUILD_TYPE=Release -DBUILD_GUI=OFF -DBUILD_CLI=ON
cmake --build . --config Release --target bht-wallet-cli
```
