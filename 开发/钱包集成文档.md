# BHTE 钱包集成文档

## 概述

BHTE 钱包模块 (`wallet/wallet.go`) 提供完整的钱包功能，支持 ECDSA、MLDSA 和 Hybrid 三种签名模式。

## 钱包模式

| 模式 | 说明 | 使用场景 |
|------|------|----------|
| `WalletModeECDSA` | 传统 ECDSA 签名 | 兼容旧系统 |
| `WalletModeMLDSA` | 纯后量子 ML-DSA 签名 | 高安全需求 |
| `WalletModeHybrid` | ECDSA + ML-DSA 双签 | 过渡期推荐 |

## 快速开始

### 创建新钱包

```go
import "bhte/wallet"

wallet, err := NewWallet(nil)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("钱包地址: %s\n", wallet.Address().Hex())
```

### 从私钥导入

```go
wallet, err := NewWalletFromKey("1a2b3c4d5e6f...", nil)
if err != nil {
    log.Fatal(err)
}
```

### 环境变量配置

```bash
# 钱包模式: ECDSA / MLDSA / Hybrid
export BHC_WALLET_MODE=Hybrid

# 启用 ML-DSA
export BHC_MLDSA_ENABLED=true

# 要求量子安全签名
export BHC_REQUIRE_QUANTUM_SAFE=false
```

## 核心功能

### 1. 签名交易

```go
// 签名转账交易
txHash, err := wallet.SignTx(
    toAddress,      // 收款地址
    1000000,        // 金额 (wei)
    20000000000,    // Gas 价格 (wei)
    21000,          // Gas 限制
    nil,            // 数据
)
```

### 2. 签名消息

```go
// EIP-191 格式签名
signature, err := wallet.SignMessage([]byte("Hello BHC"))
if err != nil {
    log.Fatal(err)
}

// 验证签名
valid := wallet.Verify(signature, []byte("Hello BHC"))
fmt.Printf("签名有效: %v\n", valid)
```

### 3. 余额管理

```go
// 设置余额
wallet.SetBalance(1000000000000000000) // 1 BHC

// 查询余额
balance := wallet.Balance()
fmt.Printf("余额: %d wei\n", balance)
```

## 钱包管理器

管理多个钱包：

```go
manager := NewWalletManager()

// 添加钱包
manager.AddWallet(wallet)

// 获取钱包
w, exists := manager.GetWallet(address)
if !exists {
    log.Fatal("钱包不存在")
}

// 设置默认钱包
manager.SetDefaultWallet(address)

// 列出所有钱包
allWallets := manager.ListWallets()
```

## 交易结构

```go
type Transaction struct {
    From     common.Address  // 发送方
    To       common.Address  // 接收方
    Value    uint64          // 金额 (wei)
    GasPrice uint64          // Gas 价格
    GasLimit uint64          // Gas 限制
    Nonce    uint64          // nonce 值
    Data     []byte          // 附加数据
    Signature []byte         // 签名
    Hash     common.Hash     // 交易哈希
}
```

### 签名交易

```go
tx := &Transaction{
    From:     wallet.Address(),
    To:       recipientAddress,
    Value:    1000000,
    GasPrice: 20000000000,
    GasLimit: 21000,
    Nonce:    0,
}

err := tx.Sign(wallet.privateKey)
if err != nil {
    log.Fatal(err)
}

// 验证签名
valid := tx.VerifySignature()
fmt.Printf("交易签名有效: %v\n", valid)
```

## API 参考

### NewWallet

```go
func NewWallet(config *WalletConfig) (*Wallet, error)
```

创建新钱包。

**参数:**
- `config`: 钱包配置，传 `nil` 使用默认配置

**返回值:**
- `*Wallet`: 新钱包实例
- `error`: 错误信息

### NewWalletFromKey

```go
func NewWalletFromKey(privateKeyHex string, config *WalletConfig) (*Wallet, error)
```

从十六进制私钥导入钱包。

**参数:**
- `privateKeyHex`: 十六进制私钥字符串
- `config`: 钱包配置，传 `nil` 使用默认配置

### Sign

```go
func (w *Wallet) Sign(data []byte) ([]byte, error)
```

对数据签名（SHA-256 哈希）。

**参数:**
- `data`: 要签名的数据

**返回值:**
- `[]byte`: 65 字节签名 (r, s, v)
- `error`: 错误信息

### SignMessage

```go
func (w *Wallet) SignMessage(message []byte) ([]byte, error)
```

EIP-191 格式签名消息。

**参数:**
- `message`: 消息内容

**返回值:**
- `[]byte`: 65 字节签名
- `error`: 错误信息

### Verify

```go
func (w *Wallet) Verify(signature, data []byte) bool
```

验证签名。

**参数:**
- `signature`: 签名
- `data`: 原始数据

**返回值:**
- `bool`: 签名是否有效

### SignTx

```go
func (w *Wallet) SignTx(to common.Address, value, gasPrice, gasLimit uint64, data []byte) (common.Hash, error)
```

签名交易并返回交易哈希。

## 安全注意事项

1. **私钥保护**: 永远不要在代码中硬编码私钥，使用环境变量或密钥管理服务
2. **量子安全**: 建议使用 `WalletModeHybrid` 模式进行过渡
3. **签名验证**: 始终验证收到的签名的有效性
4. **Nonce 管理**: 确保交易的 nonce 值正确，避免双花

## 钱包模式选择建议

| 场景 | 推荐模式 |
|------|----------|
| 现有系统集成 | ECDSA |
| 新部署、高安全需求 | MLDSA |
| 过渡期、兼容性需求 | Hybrid |
| 主网上线初期 | Hybrid |
