## 🔧 测试脚本

### 自动化测试脚本

```bash
#!/bin/bash
# run_all_tests.sh

set -e

echo "=== BHC + BHTE 自动化测试 ==="

# Phase 1: BHC 测试
echo "--- Phase 1: BHC Layer 1 测试 ---"

# 启动 BHC 节点
./build/src/bitcoind -regtest -daemon
sleep 5

# 运行 BHC 测试
./build/src/test/test_bhc --log_level=test_suite

# 运行混合签名测试
python3 ./tools/bhc_hybrid_test.py

# Phase 2: BHTE 测试
echo "--- Phase 2: BHTE Layer 2 测试 ---"

# 启动 BHTE 节点
./bhte/build/bin/geth --bhte --datadir ~/.bhte &

# 运行合约测试
cd contracts && npx hardhat test

# Phase 3: 集成测试
echo "--- Phase 3: 跨层集成测试 ---"

# 运行集成测试
python3 tests/integration/test_bridge.py

echo "=== 所有测试完成 ==="
```
