## 🔧 第一阶段：BHC内核改造&#x20;

### Step 1: ProgPoW算法集成

**目标文件：**

| 文件                       | 操作 | 说明                |
| ------------------------ | -- | ----------------- |
| `src/crypto/progpow.cpp` | 新建 | ProgPoW核心算法       |
| `src/crypto/progpow.h`   | 新建 | ProgPoW头文件        |
| `src/pow.cpp`            | 修改 | 替换SHA-256为ProgPoW |
| `src/pow.h`              | 修改 | 新增ProgPoW函数声明     |
| `src/primitives/block.h` | 修改 | 区块头结构调整           |
| `src/miner.cpp`          | 修改 | 挖矿逻辑适配            |

**关键修改点：**

```cpp
// src/pow.cpp - 需要修改的核心函数

// 原始：SHA-256工作量证明
bool CheckProofOfWork(uint256 hash, unsigned int nBits, const Consensus::Params& params);

// 改为：ProgPoW工作量证明
bool CheckProofOfWorkProgPoW(const CBlockHeader& block, const Consensus::Params& params);
```

### Step 2: ML-DSA抗量子签名

**目标文件：**

| 文件                     | 操作 | 说明                    |
| ---------------------- | -- | --------------------- |
| `src/crypto/mldsa.cpp` | 新建 | ML-DSA签名实现            |
| `src/crypto/mldsa.h`   | 新建 | ML-DSA头文件             |
| `src/script/script.h`  | 修改 | 新增OP\_CHECKSIG\_MLDSA |
| `src/pubkey.cpp`       | 修改 | 混合签名验证                |
| `src/key.h`            | 修改 | 支持ML-DSA密钥            |

**新增操作码：**

```cpp
// src/script/script.h
enum opcodetype {
    // ... 现有操作码 ...
    OP_CHECKSIG_MLDSA = 0xce,  // 新增：抗量子签名验证
    OP_L2_ANCHOR = 0xcf,       // 新增：Layer2锚定
};
```

### Step 3: 难度调整算法 (LWMA)

**替换现有DAA：**

```cpp
// 当前：比特币原始DAA（每2016块调整）
unsigned int CalculateNextWorkRequired(...);

// 改为：LWMA算法（更平滑）
unsigned int CalculateNextWorkRequiredLWMA(
    const CBlockIndex* pindexLast,
    const Consensus::Params& params
);
```

## 🛠️ 开发环境要求

### BHC (Layer 1)

- **编译器**: MSVC 2022+ / GCC 11+ / Clang 14+
- **C++标准**: C++20
- **依赖库**:
  - libkeccak (ProgPoW)
  - PQClean (ML-DSA)
  - Boost 1.80+
  - OpenSSL 3.0+

### BHTE (Layer 2)

- **Go版本**: 1.24+
- **依赖库**:
  - go-ethereum (基础)
  - bellman/arkworks (ZK证明)
  - gRPC (跨层通信)
